# Frontend - Hyunu Blog

This is the main public-facing frontend package for Hyunu Blog, built with [Next.js](https://nextjs.org).

## Getting Started

### From the Root Directory (Recommended)

Run the frontend development server from the monorepo root:

```bash
npm run dev:frontend
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### From This Package Directory

Alternatively, you can run the development server directly from this package:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## About This Package

This is the main public-facing application where visitors can read blog posts and explore content. It runs independently from the back-office administrative dashboard.
