# @hyunu/config

Shared configurations for ESLint and Prettier.

## ESLint

### For Next.js apps

```js
module.exports = require('@hyunu/config/eslint/nextjs');
```

### For Node.js apps

```js
module.exports = require('@hyunu/config/eslint/node');
```

## Prettier

```js
module.exports = require('@hyunu/config/prettier');
```
