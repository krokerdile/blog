# Back Office - Hyunu Blog

This is the administrative dashboard package for Hyunu Blog, built with [Next.js](https://nextjs.org).

## Getting Started

### From the Root Directory (Recommended)

Run the back-office development server from the monorepo root:

```bash
npm run dev:backoffice
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

### From This Package Directory

Alternatively, you can run the development server directly from this package:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

## Features

This back-office application is designed to support administrative functions independently from the main frontend, including:

- **Content Management**: Manage blog posts, pages, and media assets
- **User Administration**: Control user access and permissions
- **Analytics**: View site statistics and performance metrics
